## ADDED Requirements

### Requirement: AI 内容摘要生成
系统 SHALL 支持对文本笔记、PDF 文档和视频（基于字幕）自动生成 AI 摘要，提炼核心要点，摘要通过配置的 LLM API 生成。

#### Scenario: 生成文档摘要
- **WHEN** 用户在内容详情页点击「生成摘要」
- **THEN** 系统调用 LLM API 对内容全文进行摘要提炼，在 30 秒内返回结构化要点列表，展示在内容详情面板

#### Scenario: 视频字幕摘要
- **WHEN** 用户对已完成转写的视频点击「生成摘要」
- **THEN** 系统以字幕全文为输入调用 LLM，生成带时间节点标注的内容要点摘要

### Requirement: 关联内容推荐
系统 SHALL 在用户打开任意内容时，自动基于语义向量相似度计算并推荐最相关的 5-10 条相似内容，显示在内容详情页侧边栏。

#### Scenario: 打开笔记时推荐关联内容
- **WHEN** 用户打开一篇关于「人像后期修图」的笔记
- **THEN** 侧边栏「相关内容」区域展示语义相关的笔记/PDF/视频片段，按相似度排序

#### Scenario: 推荐结果排除当前内容
- **WHEN** 系统计算关联推荐
- **THEN** 推荐列表中不包含当前正在查看的内容本身

### Requirement: AI 题库自动生成
系统 SHALL 支持基于选定内容（笔记/文档/视频字幕）自动生成习题/考题，题型包括选择题和简答题，通过 LLM API 生成。

#### Scenario: 生成单份内容题库
- **WHEN** 用户在内容详情页选择「生成题库」
- **THEN** 系统调用 LLM 生成 5-10 道题目（含答案与解析），以可复制的格式展示

#### Scenario: 基于合集生成综合题库
- **WHEN** 用户在合集页面选择「生成题库」
- **THEN** 系统聚合合集内所有内容，生成综合性题目，可导出为 Markdown 文件

## Why

当前墨渊前端仍沿袭「科技感冷淡风」——青蓝主调、Inter 字体、dot-grid 背景，缺失文化根脉与视觉深度。施主有令：大争之世，当勇往直前！故非止于样式换皮，而当进行**结构性重构**，使赛博道士风格深入骨髓。二进制 01 与乾坤阴阳同源，数据流转即气机流布——前端界面本应彰显此层哲学。

## What Changes

**结构性变革：**
- 重构布局架构：顶部导航改为**道观经络式侧边栏**，以八卦方位划分功能区
- 重构组件结构：废弃 `.taste-*` 命名空间，全新创建 `.dao-*` 组件体系
- 重构页面组织：按「道藏、丹室、玄台」三大模块重新组织页面路由
- 重构视觉范式：从现代科技风彻底转向赛博道观美学

**视觉变革：**
- 重写 `index.css` 全局设计系统：以「墨·玄·翠玉·丹金」四色为纲，重写所有 CSS 变量
- 替换字体体系：Noto Serif SC + Geist，彻底抛弃 Inter
- 重构主题系统：默认「道」（墨色）模式，浅色模式为「阳间」
- 新增全局动效层：粒子气机背景、八卦结界装饰、灵气感应交互

**文案变革：**
- 全面术语化：所有页面、按钮、标签改用道家术语（道藏、符印、坤舆、丹室等）
- 界面文案风格化：融入偈语式表达，如「三步入道」「万象归藏」

## Capabilities

### New Capabilities

- `daoist-layout-system`：道观经络式布局体系，侧边栏导航、八卦分区、气机流动
- `daoist-component-system`：全新 `.dao-*` 组件体系（dao-card、dao-btn、dao-input、dao-dialog、dao-badge）
- `daoist-page-architecture`：三大模块架构（道藏/丹室/玄台），重构路由组织
- `daoist-motion-system`：全局动效体系（粒子气机、八卦结界、灵气感应）
- `daoist-visual-language`：四色主调、符文图标、偈语文案的完整视觉语言

### Modified Capabilities

- （无——功能行为不变，仅重构结构与视觉）
